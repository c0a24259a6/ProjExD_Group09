# アングリーこうかとん

## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
* プレイヤーのコマ,こうかとんをマウス操作で引っ張って敵のキャラ全員滅ぼすゲーム


## ゲームの遊び方
* ただのコマであるこうかとんをマウス操作で引っ張り,敵に当てることで倒せます.
* 残機が無くなるまでに全員倒せればクリア.

## ゲームの実装
### 共通基本機能
* 背景画像とキャラクター,引っ張る操作

### 分担追加機能
* ステージ追加(担当:守山):
* ステージギミック(動かせたら理想の壁)(担当:畠):
* ガイド表示(担当:国分):
* 能力(下にドッカンでしょ)(担当:片倉):
* 残機(担当:宗):
